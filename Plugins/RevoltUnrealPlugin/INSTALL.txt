If automatic plugin detection fails:

1. Create Plugins folder in your project root
2. Extract plugin files to: `YourProject/Plugins/RevoltUnrealPlugin/`
3. Edit your .uproject file and add to "Plugins" array:
   ```json
   {
     "Name": "RevoltUnrealPlugin",
     "Enabled": true
   }
   ```
4. Regenerate project files
5. Build and launch
