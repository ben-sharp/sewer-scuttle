## Usage Notes

### For RabbitCharacter Blueprint:
1. Set Skeletal Mesh to: `SK_Rabbit`
2. Set Animations:
   - Walk: `ANIM_Rabbit_Walk` or `ANIM_Rabbit_Walk_RM`
   - Jump: `ANIM_Rabbit_1Hop` or `ANIM_Rabbit_1Hop_RM`
   - Idle: `ANIM_Rabbit_IdleSatBreathe` or `ANIM_Rabbit_IdleAlertBreathe`

### For TrackPiece Blueprints:
1. Use Large Long Floor meshes for standard track pieces
2. Use Small Floor meshes for narrower sections
3. Use Crossroad Floor meshes for branching sections
4. Use Wall meshes to create lane boundaries
5. Standard track piece length: 1000 units (adjust based on mesh size)

### Recommended Track Piece Configurations:

**Basic Straight Track:**
- Floor: `SM_Large_Long_Floor_1`, `SM_Large_Long_Floor_2`, or `SM_Large_Long_Floor_3`
- Length: 1000 units

**Narrow Track:**
- Floor: `SM_Small_Floor_1` through `SM_Small_Floor_6`
- Length: 500-800 units

**Variety Track:**
- Floor: `SM_Large_Middle_Floor_1`, `SM_Large_Middle_Floor_2`, or `SM_Large_Middle_Floor_3`
- Length: 1000 units

## Revolt Plugin API Usage

To query more assets or create blueprints programmatically, use the Revolt API:

```bash
# Query assets
curl http://localhost:8080/api/search?q=Floor

# Get specific blueprint
curl http://localhost:8080/api/blueprints/BP_RabbitCharacter

# Create blueprint (requires confirm: true)
curl -X POST http://localhost:8080/api/blueprints \
  -H "Content-Type: application/json" \
  -d '{"name": "BP_MyBlueprint", "parent_class": "/Script/Engine.Actor", "confirm": true}'
```

See `Plugins/RevoltUnrealPlugin/README.md` for full API documentation.
